# Video Factory 盈利与积分方案

## 当前商业逻辑

这个项目面向 C 端用户时，不建议一开始按高价私有部署售卖。更适合先做订阅制：

- 用户注册账号后使用自己的素材、脚本和视频。
- 免费版给足一次完整体验，但保留水印和低优先级队列。
- 付费用户按月获得积分，积分用于导入、选题、脚本、分镜和渲染。
- 高成本能力按分钟扣积分，生成失败或取消时自动退回预占积分。
- 暂不接支付接口，先通过微信、支付宝、对公转账等方式收款。
- 管理员确认收款后，在后台手工切换套餐、补发积分或冻结账号。

## 套餐建议

| 套餐 | 价格 | 每月积分 | 基础模板视频 | AI 图片增强视频 | AI 图片 + 声音复刻视频 |
| --- | ---: | ---: | ---: | ---: | ---: |
| 免费版 | 0 元 | 300 | 约 3 分钟 | 约 1.6 分钟 | 约 1.3 分钟 |
| 创作者版 | 39 元/月 | 1500 | 约 15 分钟 | 约 8.3 分钟 | 约 6.5 分钟 |
| 专业版 | 99 元/月 | 5000 | 约 50 分钟 | 约 27.8 分钟 | 约 21.7 分钟 |

## 扣费规则

| 能力 | 积分 |
| --- | ---: |
| 导入资料 | 3 积分/份 |
| 自动选题 | 5 积分/次 |
| 自动脚本 | 8 积分/次 |
| 重生成分镜 | 10 积分/次 |
| 完整流水线 | 25 积分/次 |
| 基础模板视频 | 100 积分/分钟 |
| AI 图片增强视频 | 180 积分/分钟 |
| 声音复刻视频 | 150 积分/分钟 |
| AI 图片 + 声音复刻 | 230 积分/分钟 |

渲染类任务采用预占逻辑：提交时先锁定积分，渲染成功后确认扣除，失败、取消或未入队会退回。

## 管理后台必须具备

- 查看所有用户、角色、启用/停用状态。
- 查看每个用户的套餐、可用积分、已用积分、预占积分。
- 查看每个用户的导入数、脚本数、视频数、完成数和视频总时长。
- 手工切换套餐，适合线下收款后开通权益。
- 手工增减积分，适合补发积分包、退款、赠送或纠错。
- 冻结/恢复积分账户，适合异常消耗或未付款用户。
- 查看积分流水，定位每次扣费、退回和后台调整。

## 后续接支付时要补

1. 支付订单表：记录套餐、金额、订单号、支付状态和用户 ID。
2. 支付回调：微信、支付宝或 Stripe 支付成功后自动切套餐或发积分包。
3. 续费任务：每月到期后自动重置月度积分，未续费则降级或冻结。
4. 发票/收据：面向企业或工作室客户时，保留收款凭证。
5. 风控策略：高频失败、异常消耗、批量注册要进入人工审核。
